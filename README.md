# Sistema de Casa de Leilões - TDSat

## Descrição do Projeto
Este projeto consiste em um sistema informatizado desenvolvido para gerenciar as rotinas de uma casa de leilões. O software foi criado como parte das atividades de segurança e qualidade de software, focando no versionamento adequado e na organização do fluxo de trabalho.

## O que o projeto faz
O sistema permite o cadastro, listagem e gerenciamento de itens destinados a leilão, garantindo que as operações sejam registradas de forma segura e eficiente.

## Tecnologias Utilizadas
* **Linguagem:** Java
* **Banco de Dados:** MySQL (utilizando o script uc11.sql)
* **Ferramenta de Versionamento:** Git e GitHub
