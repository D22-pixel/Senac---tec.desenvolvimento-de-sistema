📚 Sistema Agência de Viagens

Atividade de Programação Orientada a Objetos - Java

🎯 Objetivo da Atividade

Desenvolver um sistema orientado a objetos em Java para uma agência de viagens que gerencie pacotes de viagem e vendas, aplicando conceitos fundamentais de POO como encapsulamento, composição e responsabilidades bem definidas.

📋 Explicação Detalhada

🏗️ Estrutura do Sistema

1. Classe Transporte

  • Atributos: tipo (String) e valor (double)
  • Função: Representa o meio de locomoção do pacote
  • Exemplos: aéreo, rodoviário, marítimo

2. Classe Hospedagem

  • Atributos: descrição (String) e valorDiaria (double)
  • Função: Representa onde o cliente ficará hospedado
  • Exemplos: "Hotel 5 estrelas", "Pousada simples"

3. Classe PacoteViagem

  • Composição: contém um Transporte e uma Hospedagem
  • Atributos adicionais: destino, quantidade de dias
  • Métodos principais:
    • calcularTotalHospedagem()
    • calcularValorComLucro()
    • calcularTotalPacote()
  • Importante: Todos os valores em dólar

4. Classe Venda

  • Atributos: nomeCliente, formaPagamento, pacote
  • Função principal: Converter valores de dólar para real
  • Método chave: converterDolarParaReal()

5. Classe Principal (Main)

  • Interface com usuário via console
  • Coleta dados e cria objetos
  • Exibe resultados formatados

🔧 Conceitos POO Aplicados

1. Encapsulamento

  • Atributos privados com métodos getter/setter
  • Controle de acesso aos dados das classes

2. Composição

  • PacoteViagem "tem um" Transporte
  • PacoteViagem "tem uma" Hospedagem
  • Venda "tem um" PacoteViagem

3. Responsabilidade Única

  • Cada classe tem uma função específica bem definida
  • Separação clara entre lógica de negócio e apresentação

4. Construtores

  • Facilitam a criação e inicialização dos objetos
  • Garantem que objetos sejam criados em estado válido

📝 Fórmulas de Cálculo

Cálculos Principais:

Total Hospedagem = valor_diaria × quantidade_dias

Margem de Lucro = subtotal × (margem_percentual ÷ 100)

Subtotal = valor_transporte + total_hospedagem

Total Pacote = subtotal + margem_lucro + taxas_adicionais

Valor em Real = valor_dolar × cotacao_dolar

🚀 Dicas de Implementação

Melhorias Sugeridas:

  1. Validações: Verificar valores negativos e dados inválidos
  2. Formatação: Usar DecimalFormat para melhor apresentação
  3. Tratamento de Erros: Implementar try-catch para entrada de dados
  4. Interfaces: Criar interfaces para diferentes tipos de pacotes
  5. Enum: Usar enum para tipos de transporte e formas de pagamento

Estrutura de Arquivos:

  • Transporte.java
  • Hospedagem.java
  • PacoteViagem.java
  • Venda.java
  • SistemaAgenciaViagens.java (classe principal)

📊 Fluxo de Execução

Sequência de Passos:

  1. Coletar dados do transporte (tipo, valor)
  2. Coletar dados da hospedagem (descrição, valor diária)
  3. Coletar dados do pacote (destino, dias, margem, taxas)
  4. Criar objetos Transporte, Hospedagem e PacoteViagem
  5. Exibir informações do pacote criado
  6. Coletar dados da venda (cliente, pagamento, cotação)
  7. Criar objeto Venda
  8. Exibir resumo completo com valores em USD e BRL
